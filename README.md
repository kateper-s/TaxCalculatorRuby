# TaxCalculator
Гем для расчета налогов (НДФЛ, налог на прибыль, НДС) с поддержкой ставок, вычетов, льгот и различных режимов налогообложения.

## Возможности

### Для физических лиц
- Расчет НДФЛ с прогрессивной шкалой налогообложения
- Поддержка стандартных, социальных, имущественных и инвестиционных вычетов
- Учет льгот (инвалиды, ветераны, многодетные, пенсионеры)
- Сравнение с альтернативными режимами (самозанятость, ИП)
- Помесячная детализация и прогнозирование налоговой нагрузки

### Для бизнеса
- Налог на прибыль (ОСН, УСН 6%/15%, ЕСХН, патент)
- Полный учет НДС с книгой покупок и продаж
- Амортизация основных средств
- Перенос убытков прошлых лет
- Расчет авансовых платежей
